# Contributing to EckeSchnecke

Thank you for your interest in contributing to EckeSchnecke! We welcome contributions from everyone.

## How to Contribute

1.  **Fork the repository** and create your branch from `main`.
2.  **Install dependencies**: `npm install`
3.  **Run tests**: `npm test`
4.  **Make your changes**.
5.  **Run tests again** to ensure nothing is broken.
6.  **Commit your changes** using conventional commits (e.g., `feat: add new feature`, `fix: resolve issue`).
7.  **Push to your fork** and submit a Pull Request.

## Development

- Run the development server: `npm run dev`
- Run tests: `npm test`
- Build the project: `npm run build`

## Code Style

We use ESLint and Prettier to maintain code quality. Please ensure your code passes linting before submitting a PR.
